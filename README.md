# Simulador Track Test - Frontend
Aplicación web de React para la simulación de un exámen.

Requisitos técnicos:

- Node (v18.14.1)
- React (v18)
- Chakra UI (https://chakra-ui.com/)
- Zustand (https://github.com/pmndrs/zustand)
